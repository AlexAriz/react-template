# react-template

This is a template structure I created to quickly start a react project.

Project structure:

```
project
│   README.md
│   .babelrc
|   index.html
|   package.json
|   webpack.config.js
│
└───src
    │
    └───assets
    |   │
    |   └───fonts
    |   │
    |   └───images
    |   |
    |   └───scss
    |
    └───js
        |   App.js
        |   index.js
        |   styles.js
        |
        └───components
        |
        └───utils
```

Libraries used:

  * React
  * Babel
  * Webpack
  * SCSS
  
Feel free to clone/download this project as a starting point for any of your own projects.
